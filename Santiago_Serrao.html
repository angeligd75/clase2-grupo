Examen Git y Github

Alumn@: Santiago Serrao

  1. Que es un sistema de control de versiones?

    Es un servidor unificado donde se guarda el codigo fuente y se puede dividir en ramas , versiones etc.

  2. Cual es la diferencia entre git y github?

    Git es el programa que se usa para esto. Github es un producto de microsoft.

  3. Cómo creo una rama en git?

    git branch y el nombre de la rama

  4. Cómo clono un repositorio?

    git clone y la url del repositorio a clonar
